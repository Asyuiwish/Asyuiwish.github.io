# Vue

# vue-cli

vue-cli是Vue.js的命令行工具，用于创建和管理Vue.js项目。它提供了一个快速的开发环境，帮助开发者快速构建项目。

# 安装vue-cli

# 安装vue-cli2.x
npm install -g vue-cli
# 安装vue-cli3.x
npm install -g @vue/cli

# 使用cli构建项目

# 构建以projectName为名的 使用webpack打包的vue项目
vue init webpack projectName

# 项目运行

npm run dev
